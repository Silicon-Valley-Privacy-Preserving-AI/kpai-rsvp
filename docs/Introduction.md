# SVAIN User Manual

**SVAIN** (Silicon Valley AI Nexus Forum) is a community platform for researchers and practitioners in privacy-preserving AI. This manual documents all features available through the web interface.

Live example can be accessed at [rsvp.k-pai.org](https://rsvp.k-pai.org).

## Sections

- [General UI](general/general.md) — Main page, navigation, theme
- [User Management](user_management/user_management.md) — Sign up and sign in
- [Seminar Management](seminar_management/seminar_management.md) — Creating, managing, and attending seminars (Staff)
- [Admin Dashboard](admin/admin.md) — User and statistics administration (Staff)

---

## Roles

| Role | Description |
|------|-------------|
| **Member** | Can browse seminars, RSVP, check in, and track attendance history |
| **Staff** | All member capabilities plus seminar creation, check-in management, CSV import, reminder emails, and admin dashboard access |
